# Agent HQ - UI/UX 改善方案

## 設計核心原則

```
Terminal 為主（≥50%）| Agent Panel 配合 Terminal 調整
```

---

## 目前痛點分析

| 問題 | 說明 |
|------|------|
| **Terminal 太窄** | 30% 寬度不夠 Claude Code 顯示資訊 |
| **全螢幕缺失** | 沒有辦法專注使用 Terminal |
| **Agent 區塊資訊不足** | 只有數量，沒有詳細 agent 卡片 |
| **不必要的收合** | Agent Panel 不需要独立收合按鈕 |

---

## 具體改進

### Terminal 區塊

| 功能 | 說明 |
|------|------|
| **三種模式** | 收合(0%) / 50% / 全螢幕(100%) |
| **切換按鈕** | Header 右側：[-] [50%] [⛶] |
| **收合狀態** | 顯示為 Header 內的小按鈕，點擊展開 |
| **字體** | JetBrains Mono 14px（可調 12-20px） |
| **行數** | 預設 30 行 |

### Agent 區塊

| 功能 | 說明 |
|------|------|
| **配合調整** | Terminal 收合 → Agent 100%；Terminal 全螢幕 → Agent 隱藏 |
| **Agent 卡片** | 顯示名稱、角色、狀態、運行時間 |
| **狀態指示** | ● 綠=運行中, ◐ 黃=思考中, ○ 灰=閒置 |
| **數量顯示** | Header 右側 badge |

### Header

| 元件 | 說明 |
|------|------|
| Logo | 左側 |
| **Project Selector** | 中央（取代 Project Selector） |
| Terminal 控制 | 右側：[-] [50%] [⛶] |

### Status Bar

| 顯示 | 說明 |
|------|------|
| **Project 名稱** | 左 |
| 連線狀態 | 中 (● 綠/○ 灰) |
| Agent 數量 | 右 |

## 寬度配置

```
┌─────────────────────────────────────────────────────┐
│                    Header (48px)                     │
├─────────────────────┬───────────────────────────────┤
│                     │                                │
│     Agent Panel     │         Terminal              │
│        50%          │           50%                 │
│                     │                                │
│        (0%)         │         (100%)                │
│                     │                                │
├─────────────────────┴───────────────────────────────┤
│                   Status (32px)                      │
└─────────────────────────────────────────────────────┘
```

- **Terminal 收合**：Agent 100%
- **Terminal 50%**：Agent 50%
- **Terminal 100%**：Agent 隱藏

---

## 移除項目

| 項目 | 原因 |
|------|------|
| Agent Panel 收合按鈕 | 多餘，透過 Terminal 控制即可 |
| New Project 按鈕 | 放入 Project Selector dropdown |
| Settings 圖示 | 目前無功能 |
| Agent 數量 badge | Status Bar 已有 |

---

## 實作優先順序

### Phase 1: Terminal 為主（1-2天）
1. 預設 Terminal 50% 寬度
2. 加入三種模式切換：[收合] [50%] [全螈幕]
3. Agent Panel 自動配合調整

### Phase 2: Agent 卡片（2-3天）
4. 顯示 agent 卡片（名稱/角色/狀態/時間）
5. 狀態顏色指示

### Phase 3: 體驗優化（3-5天）
6. 拖曳調整比例（可選）
7. 字體大小調整
8. 平滑過渡動畫
