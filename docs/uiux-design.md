# Agent HQ - UI/UX 改善方案

## 目前痛點分析

### 1. Layout 問題

| 問題 | 說明 |
|------|------|
| **Terminal 太小** | 30% 寬度不夠 Claude Code 顯示資訊 |
| **無法全螢幕** | 沒有辦法專注使用 Terminal |
| **Agent 區塊資訊不足** | 只有數量，沒有詳細資訊 |
| **分割不靈活** | 固定比例，無法根據使用者習慣調整 |

### 2. Terminal 體驗問題

- 字體太小（14px）
- 沒有 Syntax highlighting
- 沒有搜尋功能
- 沒有分頁管理

### 3. Agent 區塊問題

- 只顯示數量
- 沒有 agent 卡片
- 無法看到狀態

---

## 設計方案

### 🎯 核心原則

```
使用者體驗優先順序：
1. Terminal 操作 > Agent 觀看
2. 灵活性 > 固定配置  
3. 簡單 > 複雜
```

### 📐 推薦 Layout

```
┌─────────────────────────────────────────────────────────────────┐
│  Header (48px)                                                 │
│  [Logo] [Session▼] [New] [Settings] [Fullscreen]              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌─────────────────────────────────────────┐ ┌──────────────┐ │
│  │                                         │ │              │ │
│  │         Agent Panel                     │ │   Terminal   │ │
│  │         (可收合)                        │ │   (可全螢幕)  │ │
│  │                                         │ │              │ │
│  │  ┌─────────┐ ┌─────────┐ ┌─────────┐    │ │              │ │
│  │  │ Agent 1 │ │ Agent 2 │ │ Agent 3 │    │ │              │ │
│  │  │ ●Running│ │ ○Idle  │ │ ◐Think  │    │ │              │ │
│  │  └─────────┘ └─────────┘ └─────────┘    │ │              │ │
│  │                                         │ │              │ │
│  │  [Collapse] [Expand Fullscreen]        │ │ [⛶] [─] [×] │ │
│  └─────────────────────────────────────────┘ └──────────────┘ │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│  Status Bar (32px)                                             │
│  [Session ID] [Connected ●] [3 agents]                          │
└─────────────────────────────────────────────────────────────────┘
```

### 🔧 具體改進

#### A. Terminal 區塊

| 功能 | 說明 |
|------|------|
| **全螢幕模式** | 按鈕切換，佔滿整個畫面（扣除 Header + Status） |
| **最小化** | 可以收合成 Header 內的小按鈕 |
| **懸浮** | 全螢幕時右上角有返回按鈕 |
| **字體調整** | 使用者可調整大小（12-20px） |
| **顯示行數** | 預設 30 行，可調整 |

**推薦尺寸**：
- 預設：50% 寬度（足夠 Claude Code 顯示）
- 最小：30% 寬度
- 最大：70% 寬度
- 全螢幕：100% 寬度

#### B. Agent 區塊

| 功能 | 說明 |
|------|------|
| **Agent 卡片** | 顯示名稱、角色、狀態、運行時間 |
| **狀態指示** | ● 綠=運行中, ◐ 黃=思考中, ○ 灰=閒置 |
| **收合** | 可收合成 Header 的下拉選單 |
| **數量顯示** | Header 右上角badge |

**推薦卡片資訊**：
```┌─────────────────────────────┐
│ ● Claude (Coordinator)       │
│   Running • 5m 32s          │
│ ─────────────────────────── │
│   "正在處理..."             │
└─────────────────────────────┘```

**推薦寬度**：
- 預設：50% 寬度
- Agent 卡片：grid 排列，auto-fit，最小 200px

#### C. 收合/全螢幕邏輯

| 狀態 | Terminal | Agent Panel |
|------|----------|-------------|
| **預設** | 50% | 50% |
| **Terminal 全螢幕** | 100% | 隱藏 |
| **Agent 全螢幕** | 隱藏 | 100% |
| **兩者皆收合** | Header 按鈕 | Header 按鈕 |

#### D. 多餘設計（移除）

| 移除 | 原因 |
|------|------|
| Session Selector 在 Header | 應該在 Agent Panel 內或獨立dropdown |
| New Session 按鈕 | 可用 dropdown 取代 |
| Settings 圖示 | 目前無功能 |
| Agent 數量 badge | 已有 Status Bar 顯示 |

## 實作優先順序

### Phase 1: 基礎優化（1-2天）
1. 調整 Terminal 預設寬度 30% → 50%
2. 加入 Terminal 全螢幕按鈕
3. 修復 Agent 卡片顯示

### Phase 2: 體驗提升（3-5天）
4. Agent Panel 收合功能
5. 拖曳調整分割比例
6. 字體大小調整

### Phase 3: 進階功能（1-2週）
7. Terminal 搜尋
8. 多分頁支援
9. 歷史記錄
